XPS LEDs Recent Changes

1.1.1 (2007-03-07)
Fixed race condition which caused the LEDs to show the start color the end of a really short test run. Contributed by Nick Hofstede. (Issue 9)
1.1.0 (2007-03-03)
Added documentation and context sensitive help. (Issue 6)
Added support to preview the selected colors. (Issue 5)
Added support to choose the LEDs' brightness. (Issue 4)
Added support to pulsate (glow) LEDs during a test run. (Issue 3)
Added support to choose which LEDs should show test results. (Issue 2)
Added support to turn LEDs off (after a certain period) when a test run has finished. (Issue 1)
1.0.0 (2007-01-08)
Initial release.
