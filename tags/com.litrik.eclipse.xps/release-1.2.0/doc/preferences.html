XPS LEDs Preferences

Listen to test runs
Whether the results of JUnit test runs should be shown using the LEDs.
Colors
Start test run
Color shown when a test run starts. Click on the Preview button to preview the selected color.
Test success
Color shown when a test run completes successfully. Click on the Preview button to preview the selected color.
Test failure
Color shown when a test fails. Click on the Preview button to preview the selected color.
Brightness
Brightness of the LEDs.
Pulsate
Whether the LEDs should pulsate (glow) during the test run.
Automatically turn LEDs off
Time-out after success
Number of seconds to wait before the LEDs turn off after a successful test run. 0 means that the LEDs will not be turned off.
Time-out after failure
Number of seconds to wait before the LEDs turn off after a test run with failures. 0 means that the LEDs will not be turned off.
